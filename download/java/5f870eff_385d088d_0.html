对一组数据进行运算，针对一组数据（多行记录）只返回一个结果，也称分组函数

多行函数包含：

  函数  

  描述  

  COUNT()  

  统计表中记录的数目  

  SUM()  

  计算指定字段值的总和  

  AVG()  

  计算指定字段值的平均值  

  MAX()  

  统计指定字段值的最大值  

  MIN()  

  统计指定字段值的最小值
