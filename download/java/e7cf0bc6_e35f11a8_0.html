  6.  * 现在想要具备能力，继承一个类：Thread，具备了争抢资源的能力

  11.     但是这个方法不能是随便写的一个方法，必须是重写Thread类中的run方法
  12.     然后线程的任务/逻辑写在run方法中
  13.      */
  14.     @Override
  15.     public void run() {
  16.         //输出1-10
  17.         for (int i = 1; i <= 10 ; i++) {
  18.             System.out.println(this.getName()+i);

  20.     }
  21. }
