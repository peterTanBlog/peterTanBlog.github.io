【1】以int类型为案例：
比较的思路：将比较的数据做差，然后返回一个int类型的数据，将这个int类型的数值  按照 =0  >0  <0 

  1. int a = 10;
  2.         int b = 20;
  3.         System.out.println(a-b); // =0  >0  <0
【2】比较String类型数据：
String类实现了Comparable接口，这个接口中有一个抽象方法compareTo，String类中重写这个方法即可 

  1. String a = "A";
  2.         String b = "B";
  3.         System.out.println(a.compareTo(b));
